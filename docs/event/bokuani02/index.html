ぼくアニ

アニソンクラブイベント！ぼくアニ！！アニソンを聴きながらぼくたちで楽しもう♪

ぼくアニ第2話

ぼくたちでもアニクラできますか？から「ぼくアニ」に名前を変え、さらにパワーアップ！秋葉原MOGRAで開催です！！アットホームな雰囲気が特徴のアニソン原曲中心のDJイベントです。ぼくたちと一緒に楽しみましょう！

POINT

アットホーム

アットホームな雰囲気を大切にしているのでアニクラ初心者も安心して楽しめます。意外とお一人で来る方も多くいます。何かあればすぐにぼくアニクルーに話かけよう！

コスプレ

女子更衣室(無料)とクローク（有料）を用意しています。コスプレをしながらパーティを楽しめます。わからないことがあればコスプレスタッフが案内します。

日曜お昼に開催

13時から始まり19時に終わる。学生にも社会人にも優しい時間帯！いっぱい遊んだ後はまっすぐ家に帰るもよし！もう一軒行くのもよし！？

GUEST

  • DJ

    ありす
    (あにじゅう。/jumeaux/アニソンジャンク)
    専門学校時代にDJを独学で始め、2014年に本格的にアニソンDJを始める。感情を軸に選曲し、心地よい空間を作るのに定評がある。第5金曜日に開催中の、誰かに寄り添いたくなるアニソンメインのオールジャンルパーティー「あにじゅう。」の主催をしている。2016年冬にはアニソンクラブイベント総決算「冬の宴」にも出演。
    キュートでポップなガールズDJ B2B & FASHION unit、「jumeaux(じゅーも)」としても活動中。
    あにじゅう。
    jumeaux shop

  • DJ

    小太郎
    (でぃすあに/俺パ)
    ダンス一筋だったが好きなアニソンをダイナミックに聴きたいがためにDJ活動を始める。
    歌謡曲とロックを聞いて育ち、青春時代に血と硝煙の匂いがするアニメに影響を受けまくった結果、鋼の心臓を持つミリタリー好きおばさんに成長を遂げた。将来の夢は草薙素子。

  • VJ

    ちゃー
    (アジアニ/エロ＋アニ/リスリク/キャラSS)
    Kanonとアストラエアの聖地札幌から約300km離れた田舎から上京した、動画とお酒が大好物な引きこもり道産子。
    2015年よりVJ活動を開始。
    仮想(引きこもり)と現実(現場)を行き来しながらゆっくり成長する魚類である。※オフ時はただの飲んだくれ
