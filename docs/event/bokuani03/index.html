POINT

アットホーム

アットホームな雰囲気を大切にしているのでアニクラ初心者も安心して楽しめます。意外とお一人で来る方も多くいます。何かあればすぐにぼくアニクルーに話かけよう！

コスプレ

女子更衣室(無料)とクローク（有料）を用意しています。コスプレをしながらパーティを楽しめます。わからないことがあればコスプレスタッフが案内します。

日曜お昼に開催

13時から始まり19時に終わる。学生にも社会人にも優しい時間帯！いっぱい遊んだ後はまっすぐ家に帰るもよし！もう一軒行くのもよし！？

GUEST

にしみつ
(Xi-lium)
2000年代前半頃からDJを始める。
ふだん秋葉原MOGRAで第２日曜に行われているXi-lium（サイリウム）というアニソンDJイベントに出演してます。
好きなバックトゥザフューチャーは『1』。
ふくゐちゃん
(アニソンディスコ/アニソン侍)
幼少期より『おかあさんといっしょ』のけんたろうお兄さんや『ハッチポッチステーション』のグッチ裕三といったアーティストの影響を受けて育ち、NHKミュージックの虜となる。
『みんなのうた』の「コンピューターおばあちゃん」が収録されたカセットを擦り切れるまで聞き込み、『カードキャプターさくら』に憧れてローラーブレードを練習していた頃、『天才テレビくん』に出演していた飯田里穂ちゃんの可愛さに衝撃を受け上京を志す。
『なんでもQ』全問正解という神童ぶりを発揮しつつ、高校時代『クッキングアイドル アイ！マイ！まいん！』を心の支えとして東京大学に合格した後は、湯島の下宿を拠点として貪欲にアニメを鑑賞。最近では、『0655/2355』や『びじゅチューン！』などに惹かれつつも、「結局『忍たま乱太郎』が１番落ち着くよねー」と語る、今最もいい感じのDJの１人である。
「上手いよりも、面白い。面白いよりも、忘れられない」を目指しております！全力でいきます！！
カナタ
(アニトピア/アニソン虎ck！/げんちょうけん/スパクラ/NowPraying/GE・BO・KUナイト/ANTHEM;OZISAN/ふぁいくら/ヲタクドライブ/ちゃんたくナイト/サオアニ)
カラオケ好きが転じて2014年にDJデビュー。
その後2015年にはVJとしても活動を始め、最近ではオーガナイザーまで始めてしまった。
二次元アイドルやかわいい系、感動系のアニメを好み、重低音の効いたかっこいいアニソンも好きな雑食。DJでの経験を活かし、VJでは時に笑いを、時に涙を誘うプレイでDJと一体となってフロアを揺らす。DJ/VJとして休み無くイベントに関わり、毎週睡眠不足と落ちてゆく体力にガチンコ勝負を挑みながら全力でイベントを楽しんでいる。
