終了30分前はレイヤーさんに着替えて帰りの支度をするように声をかけて下さい。終了時刻と同時にお客さんがはけないと大変なことになります。

最後に

基本的にはこのような形で行動して下さい。色々と書きましたが、イベントやっていると時間が押したり、予想外のことが起きて、なかなか上手くいきません。
いつものことです。それを乗り越えるためには臨機応変さが必要です。持ち場が違う時間帯でも急用事は助け合いましょう。何かあればDJ/VJ陣も協力します。
逆に協力して入場、お金、荷物の管理が守られていれば、好きな曲で高まっていても問題ありません。遊びに来てくれた方ともお話をして下さい。

みんなで楽しいイベントにしましょう。よろしくお願い致します。
