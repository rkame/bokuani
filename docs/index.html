アニソンクラブイベント
ぼくアニ

奇数月の日曜日のお昼にアニソンクラブイベントを開催しています。アニソン原曲中心のDJイベントです。ぼくたちと一緒に楽しみましょう！

アットホーム

アニクラ初心者も安心して楽しめます。アニメ好きのみんなと仲良くなれるかも。意外とお一人で来る方も多くいます。ぼくアニクルーにも話かけてみよう！

コスプレ

女子更衣室(無料)とクローク（有料）を用意しています。コスプレをしながらパーティを楽しめます。わからないことがあればコスプレスタッフが案内します。

日曜お昼に開催

13時から始まり19時に終わります。学生にも社会人にも優しい時間帯！いっぱい遊んだ後はまっすぐ家に帰るもよし！もう一軒行くのもよし！？

EVENT

次回のぼくアニは7月716日曜日13:00〜19:00開催！！
豪華なゲスト、新レギュラーにキヤを迎え、「ぼくアニ」としての第１話、これまでよりパワーアップして盛り上げていきます！
